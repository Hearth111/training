# training

このリポジトリには歌練習カレンダーのサンプルサイトが含まれています。
フロントエンドはそのままに、Python (Flask) を用いた簡易サーバーを追加しました。

## 使い方

1. 依存ライブラリをインストールします。

   ```bash
   pip install flask
   ```

2. サーバーを起動します。

   ```bash
   python app.py
   ```

   デフォルトではポート `8000` 番で起動します。ブラウザで `http://<VPSのIPアドレス>:8000/` にアクセスしてください。

練習記録は `data.json`、技術テーマは `tech.json` に保存されます。
